# Planning & Requirements

This folder contains all deliverables from the Planning 
and Requirements phase of the project.

## Scope
Design a table-top assistive robotic arm for individuals 
with limited mobility to pick and place small household objects.

## Objective
- Enable independence through voice and joystick control
- Keep the solution affordable (budget: GHS 2,150 – 2,500)
- Use Arduino + Raspberry Pi as the core control system

## Budget
Total estimated cost: GHS 2,150 – GHS 2,500
